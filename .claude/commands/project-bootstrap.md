Objetivo: crear el bootstrap real del repositorio.

Pasos:
1. Leer `CLAUDE.md` y `docs/`.
2. Proponer la estructura del monorepo.
3. Crear el shell mínimo del repo con apps, packages, tooling y docs.
4. No inventar endpoints ni eventos sin documentarlos.
5. Dejar el repo arrancable en local.
6. Ejecutar lint, typecheck y tests mínimos del bootstrap.
7. Resumir lo creado y lo pendiente.
