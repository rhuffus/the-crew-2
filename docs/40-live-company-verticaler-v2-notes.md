# Verticaler v2 — Notas para el pivot

## Decisión
Verticaler debe seguir existiendo, pero ya no como una empresa totalmente sembrada desde el arranque.

## Nuevo papel de Verticaler
Verticaler pasa a ser:
- empresa viva de referencia
- demo narrativa del bootstrap CEO-first
- cobertura funcional del modo design + live
- validación de proposals, growth y runtime

## Evolución recomendada
### Antes
- seed completa desde bootstrap

### Después
- project seed
- CEO de Verticaler
- primeros objetivos
- primeras proposals
- primeros departments mínimos
- primeros teams mínimos
- especialistas solo cuando sean necesarios
- ejemplos de workflows y runtime

## Regla
Cuando cambie el pivot Live Company, revisar si Verticaler debe reflejarlo.
